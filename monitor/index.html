対応範囲（モニターでも手抜きしない範囲）

小規模Web制作の“よくある内訳”をテンプレ化して、必要な機能だけを選んで積み上げます。

基本（含まれるもの）
  • ディレクション（要件整理 / 構成提案）
  • 初期設定（GitHub Pages想定）
  • 基本デザイン調整（テンプレベースで整える）
オプション（必要な人だけ）
  • ページ追加
  • 問い合わせフォーム（GAS）
  • LINE通知 / メール通知
  • 予約機能（DB/管理 + カレンダー/時間枠）
  • ライティング（ページ単位）
  • 公開・ドメイン・SSL設定
モニター条件
  • 制作実績として掲載OK（サイトURL/キャプチャ）
  • 修正回数：原則 2回まで（軽微修正は柔軟に対応）
  • 素材提出（ロゴ/写真/文章）のご協力
対応できないこと（事故防止）
  • 複雑なEC決済（Shopify等は別提案）
  • 会員ログイン/多権限管理など大型機能
  • 大規模なSEO記事量産（別プラン）

制作の流れ（最短で迷わない）

ヒアリング→見積→提案→制作を「自動化＋テンプレ」でスムーズに進めます。

  1. フォーム回答
    要件ヒアリングをGoogleフォームで回答（5〜10分）
  2. 自動見積・提案
    回答内容から見積と提案のたたき台を自動生成
  3. すり合わせ（15〜30分）
    ズレが出やすい点だけ確認して確定
  4. 制作→確認→公開
    公開後は「改善の余地」が一番価値になります

よくある質問

モニター価格の適用条件は？

制作実績として掲載OK（サイトURL/キャプチャ）にご協力いただける方が対象です。公開後に相談したい場合もOKです（割引適用は要相談）。

特急対応（+10,000円）とは？

通常より制作優先度を上げて進行します。スケジュール状況によりお受けできない場合があります。

どれくらいの納期になりますか？

要件（ページ数/予約/通知/ライティング）で変動します。フォーム回答から目安を自動算出し、提案時にすり合わせます。

修正回数は本当に2回まで？

原則2回（構成・文章・デザイン大枠）。軽微な文言修正などは柔軟に対応します。無限修正を避けるためのルールです。

素材（写真/文章）がない場合は？

写真はフリー素材の提案も可能です。文章は「ライティング（ページ単位）」で対応できます。

注意事項（トラブル防止）
  • 内容により、モニター枠でもお断りする場合があります（要件が大きい/納期が極端に短い等）。
  • 実績掲載の範囲（URL/キャプチャ/掲載タイミング）は事前に確認します。
  • 価格はヒアリング内容により変動します（テンプレ積み上げ方式）。
